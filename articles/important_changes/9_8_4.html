    What changed?

    Threads

    We have fixed many bugs in threads over time, and they work completely now.

    • DisCatSharp.Net.Models.ThreadEditModel has a new field called PerUserRateLimit.
    • DisCatSharp.Entities.DiscordThreadChannel now supports DisCatSharp.Entities.DiscordThreadChannel#GetMemberAsync

    Components

    We support every Component available on the API.

    Discord Application Object

    DiscordApplication now includes the property fields:

    • CustomInstallUrl
    • InstallParams
    • Tags

    You can find the settings for these fields in the Discord Developer Portal.
