    DisCatSharp 9.X Documentation

    DisCatSharp (DCS) is an unofficial .NET wrapper for the Discord API which was originally a fork of DSharpPlus.
    The library has since been rewritten to fit quality and API standards as well as target wider range of .NET implementations. Furthermore this lib will be the most up-to-date.

    Getting Started

    New users will want to take a look through the articles for quick start guides, tutorials, and examples of use.
    Once you've gotten through the articles, head on over to the API Documentation for all classes and methods provided by this library.
